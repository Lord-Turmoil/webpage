- Description -

The idea of making such a game originated from the popular mobile game - Doodle Jump, developed by Lima Sky. It's quite fun but some how not available now. So I made it to practice my C++ at the same time.

Since I didn't use any image resources, so the game doesn't have pretty visual effects. However, recently, I learnt Flash a step further and get to know ActionScript 3.0, thus I remastered it. You can easily tell them apart, right? :)
